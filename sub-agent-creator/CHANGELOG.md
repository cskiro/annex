## [0.1.0] - 2025-11-02

### Added
- Initial release of sub-agent-creator skill
- Interactive sub-agent creation workflow with 5 phases
- Support for project-level and user-level agent installation
- Tool permission configuration with security-first defaults
- Model selection (inherit, sonnet, opus, haiku)
- Proactive agent description templates
- Validation and testing guidance
- Reference materials (models.yaml, tools.yaml, templates)
- Example agents (code-reviewer, debugger, data-scientist)
- Compliance with Anthropic's official sub-agent specifications
