## [0.1.0] - 2025-11-02

### Added
- Initial release of mcp-server-creator skill
- Interactive MCP server creation workflow with 7 phases
- Support for TypeScript and Python implementations
- STDIO and HTTP transport configuration
- Tool, resource, and prompt capability setup
- Claude Desktop integration with config generation
- MCP Inspector testing workflow
- Security best practices (environment variables, .gitignore)
- Template generation for common server patterns
- Comprehensive documentation and debugging guides
- Validation and quality checks
